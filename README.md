# 简介

Gitlab/JihuLab图床管理器，支持图片视频上传与删除、图片压缩后上传、相册管理、一键导出链接、多配置一键切换等功能

应用基于Flutter开发，现已支持Android平台。

- 支持上传前重命名、上传前自定义压缩图片大小
- 支持视频上传、图片批量上传
- 相册管理功能，支持预览图片、查看图片大小、导出为Markdown图片格式、HTML图片格式和原链接
- 可保存多套配置，一键切换管理不同项目

> 在寻找桌面端Gitlab/JihuLab图床管理工具？您可能会对[PicGo](https://docs.picgo.app/gui/)感兴趣！
